# Research Documentation

This folder keeps department-level documentation out of the working research
root. Generated evidence and backtest outputs still belong under
`runtime/artifacts/research/`.

## Sections

| Section | Purpose |
| --- | --- |
| `governance/` | Research pipeline controls, component boundaries, and publication rules. |
| `migration/` | Completed integration, manual migration, and decommission records. |
| `intraday_cn_futures/` | Human-readable notes from the CN-futures intraday research path. |

Private factor implementations belong in `../factors/`; reproducible portfolio
recipes belong in `../strategies/`.
